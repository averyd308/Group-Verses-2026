# 2026 Prayer Journal

A personal prayer journal website for lifting up friends in faith throughout 2026.

## Requirements

- [Node.js](https://nodejs.org) (v18 or newer recommended)

## How to Run

**Option A — Double-click:**
Double-click `start.bat` in this folder. It will install dependencies automatically on first run.

**Option B — Terminal:**
```
npm install
npm start
```

Then open your browser to **http://localhost:3000**

## Features

- All 12 people with their chosen scriptures for 2026
- Click any person's card to open their prayer page
- Write prayers for any person — your name is remembered for next time
- Avery's posts are marked with a **Prayer Leader** badge
- All prayers are saved to a local database (`prayers.db`) so they persist between sessions
- Everyone who visits can read and add their own prayers
